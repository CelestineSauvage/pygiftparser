# pygiftparser
[Gift](http://microformats.org/wiki/gift) parser in python with HTML generation. 

## Requirements

- python3
- markdown
- yattag

## Install
```
pip install -e git://github.com/CultureNumerique/pygiftparser#egg=pygiftparser
```
Import it in your application like this
```
import pygiftparser
```
